CSS Multiple Columns

The CSS multi-column layout allows easy definition of multiple columns of text.

Multi-column properties

  • column-count
  • column-gap
  • column-rule-style
  • column-rule-width
  • column-rule-color
  • column-rule
  • column-span
  • column-width

    • Example

    The column-count property specifies the number of columns an element should be divided into.The following example will divide the text in the <div> element into 3 columns:
