Styling links using css

By Default links are

  • Underlined
  • if unvisited then blue
  • Visited links are purple
  • Hovering a link makes the mouse pointer change to a little hand icon
  • Active links are red.
  • Focused links have an outline around them


    • Diffrent states of Links

  • Link: A link that has a destination
  • Visited: A link that has already been visited
  • Hover: A link that is hovered over by a user's mouse pointer
  • Focus: A link that is focused
  • Active: A link that is activated or clicked

    Mozilla Firefox
